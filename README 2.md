# Hybrid Reasoning - Agentforce Build Session

A static GitHub Pages-ready landing page for an Agentforce build session.

## Publish on GitHub Pages

1. Commit this folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Select **Deploy from a branch**.
4. Choose your branch and the root folder.
5. Open the Pages URL after deployment completes.

The site entry point is `index.html`.
